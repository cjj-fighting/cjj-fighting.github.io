React入门系列之四

Apr 07, 2019Schlägt

组件

  • render函数里面必须包含一个标签，否则会报错
  • 组件名称首字母必须大写，否则会报错
  • 页面渲染时会有一个div标签，想解决这个问题可用react(16.0版本)提供的Fragment占位符
  • 所有组件类都必须有自己的 render 方法，用于输出组件
  • class 属性需要写成 className ，label标签的for 属性需要写成 htmlFor ，这是因为 class 和 for 是 JavaScript 的保留字
组件传值

子组件向父组件传值：

  • 父组件向子组件传一个方法handleMethods，接受子组件传递的参数
  • 子组件通过this.props.handleMethods调用父组件的方法，同时向父组件传递参数

父组件向子组件传值：

  • 父组件向子组件传递一个属性值
  • 子组件调用this.props.属性值父组件的属性值
